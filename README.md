🛒 Amazon Clone (HTML & CSS)

A responsive Amazon Clone UI built using only HTML and CSS, inspired by the design of Amazon.
This project focuses on frontend design, layout structuring, and responsive styling without using JavaScript.

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & Layout
Flexbox
Grid
Media Queries

✨ Features
🏠 Amazon-style homepage layout
🧭 Navigation bar with logo, search bar & menu
🛍️ Product sections with cards
📸 Banner / hero section
📱 Fully responsive design (mobile-friendly)
🎨 Clean UI using pure CSS
